Perfect for: Sales calls, customer support, team meetings, client presentations, and any professional video conversation.

Troubleshooting

Installation Issues

Windows Security Warning

  • Click "More info" then "Run anyway"
  • This is normal for new applications
  • TranzyNote is completely safe and malware-free

Installation Won't Start

  • Right-click installer and select "Run as Administrator"
  • Temporarily disable antivirus software
  • Ensure you have admin rights on your computer

Audio Problems

Microphone Not Working

  • Go to Windows Settings → Privacy → Microphone
  • Allow apps to access your microphone
  • Select the correct microphone in TranzyNote settings
  • Restart the application after changing permissions

Can't Hear Audio Playback

  • Check your speaker settings in Windows
  • Ensure audio capture is enabled in TranzyNote
  • Test with headphones for better audio quality

Application Issues

Overlay Not Staying on Top

  • Restart TranzyNote
  • Disable "Focus Assist" in Windows settings
  • Close fullscreen apps and try again

Overlay Not Hiding During Screen Share

  • Wait 2 seconds for auto-detection
  • Manually hide using keyboard shortcut
  • Update to the latest version
